INFO ABOUT THE SITE

DESCRIPTION

This web application is part of my final project for my computer engineering degree. Its objective is to create a site where you can search for videogames and easily keep track of them. You can also see the last news about the world of videogames.

To perform the querys I have made use of the RAWG Games API. IT is a very reliable open data source with many search options and a large amount of avilable metadata.

  • - RAWG Video Games Database API URL: https://rawg.io/

I've also used the NewsAPI to retrieve the latest news about the world of videogames.

  • - News API URL: https://newsapi.org/
