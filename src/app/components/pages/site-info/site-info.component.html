DESCRIPCIÓN

Esta aplicación web forma parte de mi proyecto final de carrera de ingeniería informática. El objetivo de la misma es crear un sitio en el que poder buscar videojuegos y hacer un seguimiento de estos de forma sencilla. También puedes estar al día de las últimas noticias sobre el mundo de los videojuegos.

Para poder hacer las búsquedas he hecho uso de la API de RAWG Games. Es una fuente de datos abierta muy fiable con muchas opciones de búsqueda y una gran cantidad de metadatos disponibles.

  • - URL de RAWG Video Games Database API: https://rawg.io/

También he hecho uso de la API de NewsAPI para poder consultar las últimas noticias sobre el mundo de los videojuegos.

  • - URL de News API: https://newsapi.org/
